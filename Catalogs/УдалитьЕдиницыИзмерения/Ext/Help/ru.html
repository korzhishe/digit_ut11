(Не использовать) Удалить единицы измерения

Справочник предназначен для регистрации перечня единиц измерения, определенных для позиций номенклатуры (товаров или услуг).
Каждая позиция характеризуется базовой единицей измерения. При необходимости для номенклатуры могут быть назначены упаковки - альтернативные единицы измерения, для каждой из которых определяется коэффициент пересчета относительно базовой единицы (переключатель Соcтоит из других упаковок).

Если товары могут поступать и отгружаться в одинаковых упаковках, то для таких товаров предусмотрено создание набора упаковок.
