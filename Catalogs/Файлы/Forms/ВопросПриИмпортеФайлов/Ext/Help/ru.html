Программа выводит список файлов, которые невозможно загрузить в программу, так как они превышают максимальный размер.

В программу можно загрузить несколько файлов.

Нажмите Продолжить, для того чтобы:

  • загрузить остальные файлы из перечисленных в списке для загрузки, которые проходят по размеру;
  • получить отчет об ошибках с перечнем файлов, по которым импорт не состоялся.

Нажмите Отмена для возврата в программу без продолжения попыток. Если в списке для загрузки было перечислено несколько файлов и другие файлы удовлетворяли условиям, они также не будут загружены в программу.

Предельно допустимый размера файлов можно увеличить в настройках программы. Для этого в разделе Администрирование - Настройки работы с файлами укажите Максимальный размер файла. По умолчанию равен 50 Мб.
