Многофункциональная обработка отчет по анализу цен.

Показатели отчета:

Пометка - Если флаг установлен, позиция будет перенесена в документ Установка цен номенклатуры.  

Количество остаток - текущий остаток по всем складам.

Количество приход - количество поступившее за период.

Цена поступления - цена из приходной накладной. Если отчет выводится по номенклатуре в целом, в колонку выводится максимальная цена из всех поступлений. На закладке Настройка можно установить флаг вывода минимальной цены. В таком случае можно получить например информацию по минимальной закупочной цене, для дальнейшей установки её как цены компании. Может служить для плановой себестоимости при оформлении заказов поставщикам.

Изменение закупочной цены % - соотношение цена поступления+доп.расходов с той ценой которая установлена как закупочная.

Средняя цена остатков - средняя себестоимость единицы остатков. Расчитывается как себестоимость всех партий товара на остатке / количество.

Старая базовая цена - закупочная цена зафиксированная в базе на дату отбора цен (по умолчанию вчерашний день).

Новая базовая - последняя закупочная цена в базе. В случае если обработка сделана на основании нового поступления, выводится цена из приходной накладной.

Старая наценка% - соотношение цены старой цены закупки к старой цене продажи (историческая)

Наценка новая % - соотношение новой закупочной цены к новой цене продажи (при заполнении отчета новая наценка устанавливается равной старой наценке, подразумеваем сохранение наценки).

Старая расчетная цена - цена продажи которая числится в базе.

Новая расчетная цена - цена продажи на основании новой закупочной цены с наценкой.

Вспомогательные окна анализа:

1) вывод истории по всем поступлениям данного товара за весь период. Выводится: дата, документ, количество по документу, цена в документе, поставщик. В табличку выводятся все поступления

по текущей позиции.

2) вывод истории изменения закупочной цены числящейся в базе.

3) вывод закупочных цен разных поставщиков. Берется из регистра Цены номенклатуры контрагентов. Выводятся последние числящиеся значения.

Функции:

- Установка цен. Создание документа Установка цен номенклатуры с переносом указанных цен и товаров в новый документ. При нажатии откроется новый заполненный документ.

- В колонке "Новая наценка" можно задавать свою наценку как для каждого товара, так и на конкретные папки. В случае если наценка установлена на папку обработка установит наценку и расчитает новые цены для всей папки номенклатуры.

Закладка настройка:

Документ для анализа. Быстрый выбор документа для анализа. При выборе документа в отбор автоматически заполнится номенклатура указанная в табличной части "Товары". Может служить для анализа конкретного заказа поставщику (например когда в базу был заведен полученный счет от поставщика с его ценами, для того чтобы сравнить цены поставщика с минимальными закупочными ценами компании), поступления, оприходования (для получения информации по какой цене приходовать товар)

Отборы: Номенклатура, Поступления. В отборах можно выбирать любые поля номенклатуры или поступлений.

Пример №1: можно выбрать папку номенклатуры и задать условие по дате поступления. В отчет сформируется вся указанная номенклатура. При этом в колонке цена закупки автоматически установится максимальная цена которая была из приходных накладных,  колонка Поступление количество - выводит приход за весь период.

Пример №2: можно поставить условие по текущей дате, таким образом получаем отчет за день по всем приходным накладным.
