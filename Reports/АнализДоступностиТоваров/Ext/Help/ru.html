(не используется) Доступные для продажи товары

Отчет предназначен для контроля состояний запасов товаров и анализа общего и доступного остатка товаров.

Особенности отчета:

  • В наличии на складе - общее количество товаров.
  • Отгружается - количество товаров, уже запланированных к отгрузке по документам Реализация товаров.
  • Резерв - количество зарезервированное по заказам. При варианте контроля остатков С учетом графика отображается количество запланированное к отгрузке до границы контроля графика.
  • Обособленный резерв - количество товаров по обособленно обеспечиваемым заказам.
  • Свободно для продажи - остаток на складе за минусом количества из граф Недоступно для продажи.
