(не используется) Выписка по расчетному счету

Документ предназначен для отражения в системе факта поступления или списания денежных средств с расчетных счетов предприятия.

В документ включаются все безналичные платежи, которые были оформлены в системе соответствующими документами Списание безналичных денежных средств и Поступление безналичных денежных средств.
Выписка составляется за каждый операционный день, в котором банком были произведены операции со счетом организации.

Рабочее место представлено в виде закладок:

  • На закладке Выписки из банка представлен список банковских выписок (документов Выписка по расчетному счету). Документы представлены в списке в хронологической последовательности.
  • На закладке Входящие платежи представлен общий список входящих платежей, введенных в систему. Для тех платежей, которые уже зарегистрированы в банковской выписке установлен флаг Проведено банком.
  • На закладке Исходящие платежи представлен общий список исходящих платежей, введенных в систему. Для тех платежей, которые уже зарегистрированы в банковской выписке, будет установлен флаг Проведено банком.


Закладки Входящие платежи и Исходящие платежи предназначены для проверки соответствующих видов оформленных платежей, которые еще не отражены в банковской выписке. С помощью группы команд Создать на основании можно добавить выделенные платежи в уже созданную выписку или создать новую выписку.

При работе по системе Клиент-банка документ Выписка по расчетному счету создается автоматически на основании поступившей от банка информации о проведенных платежах.

Контроль и анализ оперативной информации по всем движениям безналичных денежных средств осуществляются с помощью отчетов Остатки денежных средств. Для контроля операций с денежными средствами предусмотрен отчет Контроль операций с денежными средствами.
