Установки скидок (наценок)

Документ Установка скидок (наценок) предназначен для ввода в действие скидок (наценок). Скидки (наценки), указанные в документе действуют как при продаже товаров оптовым клиентам, так и при продаже товаров в розницу с помощью документа Чек ККМ.

Документ доступен при условии включенной функциональной опции Скидки наценки при продаже (Администрирование/Маркетинг и планирование).

Указание маркетингового мероприятия не является обязательным, документ Установка скидок (наценок) может быть оформлен без указания маркетингового мероприятия.

Список скидок (наценок) указывается в табличной части документа . Для каждой скидки (наценки) указывается период ее действия и склад, при оформлении продаж с которого будет действовать скидка (наценка). В качестве склада могут быть указаны как оптовые склады, так и склады типа Розничный магазин.

Важно. Для одного склада (розничного магазина) можно указать несколько различных скидок (наценок). В этом случае скидки (наценки) будут действовать в соответствии с теми правилами взаимодействия, которые установлены в списке скидок (наценок).

Важно. При оптовой торговле действие скидки (наценки) может быть ограничено не только конкретным складом, но и сегментом партнеров. При оптовой торговле скидки (наценки) по маркетинговым мероприятиям применяются совместно с теми скидками (наценками), которые определены в соглашении с клиентом. Правила совместного применения скидок(наценок) определяются в списке скидок (наценок).

Как создать новый документ Установка скидок по маркетинговым мероприятиям?
  • В списке документов Установка скидок по маркетинговым мероприятиям нажмите на кнопку Создать.
  • Будет создан новый документ Установка скидок по маркетинговым мероприятиям.
  • В новом документе необходимо указать маркетинговое мероприятие и ввести список скидок (наценок), которые будут действовать в рамках маркетингового мероприятия.
